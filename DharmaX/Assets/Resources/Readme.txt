1. Add a new layer named GUI(Edit-project settings-tags)
2. Uncheck created GUI layer from your Main Camera culling mask.
3. Simply drag the GUI prefab anywhere on your scene. (You can find this prefab here: Resources/Holographic/prefab) 
4. Verify that the GUI Camera has GUI layer checked in camera culling mask(It should be the only checked element in GUI Camera culling mask!)
5. Your main camera have a different name simply open a grid.cs script in a scripts folder and change "Main Camera" to your camera name
And that's all!
